# 📱 تقرير تطوير تطبيق Global Hedge AI للهاتف المحمول

## 🔧 **كيفية التشغيل**

### **1. تثبيت التبعيات:**
```bash
cd apps/mobile
npm install
```

### **2. تشغيل التطبيق:**
```bash
# تشغيل مع Expo Go
npm start

# تشغيل على Android
npm run android

# تشغيل على iOS (macOS فقط)
npm run ios

# تشغيل على الويب
npm run web
```

### **3. البناء للنشر:**
```bash
# بناء APK للـ Android
npx expo build:android

# بناء للـ iOS
npx expo build:ios
```
